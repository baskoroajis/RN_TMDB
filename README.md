# RN-Starter-And-Components
This repository contains starter project and boilerplate components for react native

# Install dependency
- npm install
- cd ios && pod install

# How To run 
- npm run android
- npm run ios

# please change The Movie Database Key with your own
- const MOVIEDB_KEY in Api.js
