# RN-Starter-And-Components
This repository simple TMDB (https://www.themoviedb.org/) APP developed using react native

# Install dependency
- npm install
- cd ios && pod install

# How To run 
- npm run android
- npm run ios

# Please change TMDB Key with your own
- const MOVIEDB_KEY in Api.js
